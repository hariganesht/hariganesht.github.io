Variational Inference from Scratch: ELBO, Mean-Field, and Why It Works

1. The problem with exact inference

Bayesian inference requires computing the posterior $p(\mathbf{z} \mid \mathbf{x})$ over latent variables $\mathbf{z}$ given observations $\mathbf{x}$. By Bayes' theorem:

$$ p(\mathbf{z} \mid \mathbf{x}) = \frac{p(\mathbf{x} \mid \mathbf{z})\, p(\mathbf{z})}{p(\mathbf{x})} $$

The denominator, the marginal likelihood $p(\mathbf{x}) = \int p(\mathbf{x} \mid \mathbf{z})\, p(\mathbf{z})\, d\mathbf{z}$, is the problem. For all but the simplest models (conjugate families), this integral is intractable. Variational inference replaces this integration problem with an optimization problem.

2. KL divergence

The Kullback–Leibler divergence from a distribution $q$ to a distribution $p$ measures how much information is lost when $q$ is used to approximate $p$:

$$ \KL(q \| p) = \E_{q(\mathbf{z})}\!\left[\log \frac{q(\mathbf{z})}{p(\mathbf{z})}\right] = \int q(\mathbf{z}) \log \frac{q(\mathbf{z})}{p(\mathbf{z})}\, d\mathbf{z} \tag{1} $$

Two key properties: $\KL(q \| p) \geq 0$ always (by Jensen's inequality), with equality iff $q = p$ almost everywhere. Note it is asymmetric: $\KL(q\|p) \neq \KL(p\|q)$ in general. This asymmetry has important consequences — $\KL(q\|p)$ is zero-avoiding (it penalizes placing mass where $p$ is zero), while $\KL(p\|q)$ is mass-covering.

3. Deriving the ELBO

We want to find an approximation $q(\mathbf{z})$ from some tractable family $\mathcal{Q}$ that is close to the true posterior $p(\mathbf{z} \mid \mathbf{x})$. We minimize $\KL(q \| p(\cdot \mid \mathbf{x}))$:

$$ \KL\!\left(q(\mathbf{z}) \,\|\, p(\mathbf{z} \mid \mathbf{x})\right) = \E_q\!\left[\log q(\mathbf{z})\right] - \E_q\!\left[\log p(\mathbf{z}, \mathbf{x})\right] + \log p(\mathbf{x}) \tag{2} $$

Rearranging (and noting $\log p(\mathbf{x})$ doesn't depend on $q$):

$$ \log p(\mathbf{x}) = \underbrace{\E_q\!\left[\log p(\mathbf{z}, \mathbf{x})\right] - \E_q\!\left[\log q(\mathbf{z})\right]}_{\mathcal{L}(q) \;=\; \text{ELBO}} + \underbrace{\KL\!\left(q \,\|\, p(\cdot \mid \mathbf{x})\right)}_{\geq\, 0} \tag{3} $$

Since $\KL \geq 0$, the ELBO $\mathcal{L}(q)$ is a lower bound on the log marginal likelihood $\log p(\mathbf{x})$. Maximizing $\mathcal{L}(q)$ simultaneously minimizes the KL and tightens the bound. We can also write the ELBO as:

$$ \mathcal{L}(q) = \underbrace{\E_q\!\left[\log p(\mathbf{x} \mid \mathbf{z})\right]}_{\text{reconstruction}} - \underbrace{\KL\!\left(q(\mathbf{z}) \,\|\, p(\mathbf{z})\right)}_{\text{complexity}} \tag{4} $$

This decomposition is intuitive: the first term rewards the variational posterior for explaining the observations; the second term penalizes it for being too different from the prior.

Connection to VAEs. Equation (4) is exactly the objective of a Variational Autoencoder (Kingma & Welling, 2014). The encoder $q_\phi(\mathbf{z} \mid \mathbf{x})$ is the variational posterior, the decoder $p_\theta(\mathbf{x} \mid \mathbf{z})$ is the likelihood, and the ELBO is what you maximize via the reparameterization trick.

4. Mean-field variational inference

To make the optimization tractable, we restrict $q$ to a fully factored (mean-field) family:

$$ q(\mathbf{z}) = \prod_{j=1}^{m} q_j(z_j) \tag{5} $$

Each factor $q_j$ is optimized independently. Under the mean-field assumption, the ELBO decomposes, and the optimal $q_j$ is given by:

$$ \log q_j^*(z_j) = \E_{-j}\!\left[\log p(\mathbf{z}, \mathbf{x})\right] + \text{const} \tag{6} $$

where the expectation $\E_{-j}[\cdot]$ is taken over all factors except $q_j$. This is the key result of mean-field VI: the optimal factor for $z_j$ is an unnormalized exponential of the expected log joint, averaged over the other factors.

5. Coordinate ascent VI (CAVI)

Equation (6) suggests an algorithm: cycle through each factor $q_j$, update it to $q_j^*$ using the current estimates of the other factors, and repeat until convergence. This is Coordinate Ascent Variational Inference: 

while not converged:
    for j in 1, ..., m:
        q_j ← exp( E_{-j}[ log p(z, x) ] )   # eq. (6)
        q_j ← q_j / Z_j                        # normalize
    compute ELBO; check convergence

Under the mean-field assumption, each step is guaranteed to increase (or leave unchanged) the ELBO. CAVI therefore converges to a local optimum.

6. A minimal Python implementation

Below is a minimal CAVI implementation for a Gaussian mixture model with $K$ components, $N$ observations, and mean-field variational posterior. I'm using NumPy only. 

import numpy as np
from scipy.special import digamma, logsumexp

def cavi_gmm(X, K, n_iter=100, seed=0):
    """
    Mean-field VI for a Gaussian mixture model.
    X: (N, D) observations
    K: number of components
    Returns: phi (N x K responsibilities), mu (K x D means)
    """
    rng = np.random.default_rng(seed)
    N, D = X.shape

    # Initialize responsibilities uniformly + noise
    phi = rng.dirichlet(np.ones(K), size=N)      # (N, K)
    elbo_history = []

    for _ in range(n_iter):
        # ---- M-like step: update component parameters ----
        Nk = phi.sum(axis=0) + 1e-10             # (K,)
        mu = (phi.T @ X) / Nk[:, None]           # (K, D)
        sigma2 = np.ones(K)                       # fixed variance (simplification)

        # ---- E-like step: update responsibilities ----
        log_rho = np.zeros((N, K))
        for k in range(K):
            diff = X - mu[k]                      # (N, D)
            log_rho[:, k] = (
                -0.5 * np.sum(diff**2, axis=1) / sigma2[k]
                - 0.5 * D * np.log(2 * np.pi * sigma2[k])
                + np.log(Nk[k] / N)
            )
        log_phi = log_rho - logsumexp(log_rho, axis=1, keepdims=True)
        phi = np.exp(log_phi)

        # ---- ELBO (approximate) ----
        elbo = (
            np.sum(phi * log_rho)
            - np.sum(phi * log_phi)           # entropy
        )
        elbo_history.append(elbo)

    return phi, mu, elbo_history

7. Beyond mean-field

Mean-field VI is fast and scalable but has well-known failure modes. Because we minimize $\KL(q \| p)$, the variational posterior tends to be underconfident and can miss modes of the true posterior entirely. Alternative approaches include:

  • Stochastic VI (SVI) — use minibatches and stochastic gradients to scale to large datasets (Hoffman et al., 2013).
  • Normalizing flows — transform a simple base distribution through a series of invertible maps to get a flexible $q$.
  • Monte Carlo VI / ELBO gradients — use the reparameterization trick or score-function estimators to compute gradients through $\mathcal{L}(q)$. This is what Pyro and NumPyro do.
  • Sequential Monte Carlo / importance sampling — when you need exact (asymptotically) samples rather than an approximate posterior mean.

The code above is simplified for clarity. A production implementation would track ELBO convergence more carefully and handle numerical issues. See my GitHub for cleaner versions.
